# 青年留學貸款

> **資料來源**：https://youth.gov.taipei/services/05eb79e5-46b5-41e5-b1fd-0519d782b04a/fdd64f80-8a22-4a19-b33c-004ee65db163
> **頁面更新日期**：2026-02-12
> **擷取日期**：2026-04-25

---

## 目的

協助青年克服經濟困難，赴海外深造，修讀碩、博士學位或專業、技術證照，拓展國際視野，增進學能。

> 臺北市青年留學生就學貸款補助辦法已於 **114 年（2025）9 月 1 日**更新上路。

---

## 申請資格

申請人需同時符合以下所有條件：

1. 中華民國國民，申請貸款日前已**設籍臺北市一年以上**
2. **20 歲以上未滿 46 歲**
3. 具完全行為能力及**無不良信用紀錄**
4. 申請人或受扶養人之納稅義務人，最近一年綜合所得淨額**未達課稅級距 30% 或未達申報標準**
5. 我國政府立案之高中或大專校院畢業，或具同等學力
6. 出國**全時修讀**碩、博士學位或專技證照（**不含大陸、香港、澳門及其外設地區**）

### 不得申請情形

- 貸有教育部留學生就學貸款或政府就學貸款尚未結清
- 正在領受公費或留學獎助學金期間尚未結束

## 貸款條件一覽

### 博士

| 項目 | 內容 |
|------|------|
| 名額 | 1,000 名（碩博士共用名額） |
| 貸款額度 | 最高 **250 萬元** |
| 撥款方式 | 第 1 年上限 100 萬；第 2 年累計上限 180 萬；第 3 年累計上限 250 萬 |
| 利率（前四年） | 郵政儲金一年期定儲機動利率 **＋ 1.1%** |
| 利率（第五年起） | 郵政儲金一年期定儲機動利率 **＋ 0.9%** |
| 貸款期限 | 最長 **16 年** |
| 寬限期 | 最長 **5 年**（僅付息，不還本；自第一次撥款日起算） |
| 開始還款 | 寬限期屆滿後按年金法本息均攤 |
| 手續費 | 免收 |
| 提前還款違約金 | 無 |
| 保證人 | 需另覓配偶以外 1 人，具中華民國國籍且無不良信用紀錄 |

### 碩士

| 項目 | 內容 |
|------|------|
| 名額 | 1,000 名（碩博士共用名額） |
| 貸款額度 | 最高 **120 萬元** |
| 撥款方式 | 第 1 年上限 75 萬；第 2 年累計上限 120 萬（一年制者上限 75 萬） |
| 利率（前二年） | 郵政儲金一年期定儲機動利率 **＋ 1.1%** |
| 利率（第三年起） | 郵政儲金一年期定儲機動利率 **＋ 0.9%** |
| 貸款期限 | 最長 **10 年** |
| 寬限期 | 最長 **3 年** |
| 開始還款 | 寬限期屆滿後，第 4 年起開始償還本金 |
| 手續費 | 免收 |
| 提前還款違約金 | 無 |
| 保證人 | 同博士 |

### 專技證照

| 項目 | 內容 |
|------|------|
| 名額 | **200 名** |
| 貸款額度 | 最高 **120 萬元** |
| 撥款方式 | 同碩士（第 1 年上限 75 萬；第 2 年累計上限 120 萬） |
| 利率 | 同碩士 |
| 貸款期限 | 最長 **10 年** |
| 寬限期 | 最長 **3 年** |
| 手續費 | 免收 |
| 提前還款違約金 | 無 |

## 利息補貼

本貸款**前十年（含寬限期）**之利息，由臺北市政府依規定補助：

| 情況 | 補助幅度 |
|------|----------|
| 綜合所得稅稅率未達 20%，或未達申報標準 | **全額補助** |
| 綜合所得稅稅率為 20% | **半額補助** |
| 稅率達 30% 以上（每年七月定期查核） | **不予補助** |

- 第 11 年起利息由借款人**自行負擔**
- 若申請人在補助期間將**戶籍遷出臺北市**（非因就讀本貸款核定學位），將停止補貼並須返還自遷出日起之已補貼利息
- 因修讀本貸款學位而出境 2 年以上被逕為遷出者，畢業或取證後 3 個月內應將戶籍遷回臺北市

## 應備文件

| # | 文件名稱 |
|---|---------|
| 1 | 申請人及保證人之身分證、印章 |
| 2 | 申請日前三個月內核發之戶籍謄本或戶口名簿影本（個人記事欄不可省略） |
| 3 | 有效護照影本及我國政府立案之高中或大專校院畢業證書影本 |
| 4 | 稅捐稽徵機關核定之最近一年綜合所得稅核定通知書（或二年所得資料清單） |
| 5 | 國外學校入學許可/在學證明（含護照英文全名、系所、學位、修業年限起迄） |
| 6 | 就讀學校教育部核可名冊（碩博士） |
| 7 | 留學國政府認可機構之學習證明文件（專技證照） |
| 8 | 公證授權書（申請人已在海外者，須由公證機構或我國駐外館處認證） |
| 9 | 入出國日期證明書（已出國就讀者，近三個月內由內政部移民署核發） |
| 10 | 留學獎助學金核定期間已結束之證明（如曾受領） |
| 11 | 臺北市青年留學生就學貸款專用切結書（文件編號 DE23） |
| 12 | 銀行法第 33 條之 3「同一關係人」資料表（借款總額 100 萬以下免附） |
