# 青創貸款及補助

> **資料來源**：https://youth.gov.taipei/services/05eb79e5-46b5-41e5-b1fd-0519d782b04a/a2bb5667-76c8-49c4-b518-b94f10d331df

> **擷取日期**：2026-04-25

---

## 概述

為加強協助臺北市青年創業，青年局受理青年創業融資貸款。服務方向分為兩個階段：

- **創業前**：主動走進校園，與中央單位及本市大專院校合作，透過競賽協助校園創業提案在臺北媒合場域、落地實踐，並給予相關支持資源。
- **創業初期**：從青年最缺乏的「資金」、「行銷」及「人力」三大面向協助：
  - 資金面：提供青創貸款、利息補助、共享空間租賃補助
  - 行銷面：鼓勵並輔導青年參與展會及產業媒合活動
  - 人力面：結合青年實習資源，補助創業初期青年開設實習職缺

## 青創貸款－簡易貸

### 申請對象

| 條件 | 說明 |
|------|------|
| 代表人年齡 | 已成年至未滿 46 歲，且設籍本市一年以上之中華民國國民 |
| 創業輔導課程 | 申請日前三年內曾參與政府創業輔導相關課程達 **20 小時以上** |
| 事業登記 | 依法完成登記未滿五年，且登記地址須位於本市（符合中小企業認定標準） |

### 貸款用途

以購置廠房、營業場所、機器、設備、**營運週轉金**或**創業準備金**為限。

### 貸款額度

最高 **新臺幣 120 萬元**

---

## 青創貸款－一般貸

### 申請對象

與簡易貸相同（代表人年齡、課程時數、事業登記條件）。

### 貸款用途

以購置廠房、營業場所、機器、設備或**營運週轉金**為限（不含創業準備金）。

### 貸款額度

| 情況 | 額度上限 |
|------|----------|
| 基本額度 | 最高 **200 萬元** |
| 登記未滿一年者，另可申請創業準備金 | 另加最高 **100 萬元** |
| 符合加碼條件者 | 最高 **400 萬元** |

**加碼條件**（申請時起前二年符合其一）：

1. 獲政府政策性貸款達 100 萬元以上，且依約連續攤還達 18 個月以上
2. 申請人或代表人獲本府公告之政府相關創新獎項或政府相關補助
3. 平均年營業額達 500 萬元以上

## 應備文件清單

### 基本文件

| # | 文件名稱 |
|---|---------|
| 01 | 貸款申請書及事業計畫書 1 份（簡易貸免填第十一項） |
| 02 | 代表人國民身份證正反面影本 1 份 |
| 03 | 代表人申請時三個月內之電子戶籍謄本 |
| 04 | 切結書 1 份 |
| 05 | 公司設立登記表、商業登記抄本或有限合夥登記證明影本（含最新異動資料） |
| 06 | 事業近二年資產負債表、損益及稅額計算表（須蓋國稅局收件章） |
| 07 | 事業近一年營業稅報稅資料（401/403 表，須蓋國稅局收件章） |
| 08 | 三年內政府創業輔導課程結業證明（20 小時以上） |
| 09 | 加盟契約影本（非加盟者免附） |
| 10 | 特殊條件證明文件（申請 200 萬以上者必附） |
| 11 | 事業、代表人及配偶之聯徵中心查詢授權書 |
| 12 | 承貸銀行出具攤還一年繳款證明（再次申貸者附） |

### 提供擔保品時須加附

- 建物/土地權狀或謄本影本及照片
- 機器設備等動產之產權證明文件及照片

### 自行增加保證人時須加附

- 保證人身分證正反面影本
- 保證人在職證明正本或財力證明文件
- 保證人聯徵授權書

## 申請重要須知

1. 申請前請詳閱「臺北市青年創業融資貸款實施要點」
2. 申請文件請以 A4 單面印製，依照表單順序夾妥
3. 申請所用印之企業大小章須為設立登記之印鑑章
4. **股份有限公司且法人股東占比 20% 以上者，應附公文函、登記表、法人公司章程、法人公司股東名冊**
5. 所有送件資料均不予退還
6. 線上申辦仍需郵寄紙本文件至青年局
