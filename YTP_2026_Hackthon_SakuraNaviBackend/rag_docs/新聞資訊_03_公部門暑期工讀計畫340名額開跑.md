# 最新公告｜前進市政第一線！臺北市大專青年學生公部門暑期工讀計畫開跑 提供340個名額 帶青年開啟職涯初體驗

**發布日期：** 2026-03-30
**更新日期：** 2026-03-31
**發布單位：** 職涯支援科
**來源連結：** https://youth.gov.taipei/announcement/news/d74cea04-8ac3-4cc5-9bd7-39f4bed97081

---

## 計畫概要

臺北市政府推出「大專青年學生公部門暑期工讀計畫」，提供設籍臺北市的大專院校學生深入市府第一線的機會，今年預計提供 **340個工讀名額**，邀請青年把握機會、踏出第一步。

---

## 重要資訊一覽

| 項目 | 內容 |
|------|------|
| 工讀期間 | 2026年7月1日至8月31日 |
| 工作時數 | 全職，每日8小時、周休二日 |
| 月薪 | **29,500元** |
| 工作地點 | 臺北市政府各機關單位 |
| 報名方式 | 全線上申請 |
| 報名期間 | 115年3月30日上午10時 ～ 4月17日止 |
| 錄取名單公告 | 6月5日 |
| 正式報到 | 7月1日 |

---

## 名額分配

| 提供單位 | 名額 | 條件 |
|----------|------|------|
| 臺北市政府青年局（就業安定基金補助） | 203名 | 僅限歷年**未曾**錄取本府暑期工讀者 |
| 臺北市政府人事處 | 137名 | 僅限**曾**錄取者；其中10%（14名）保留予原住民身分學生 |
| **合計** | **340名** | |

---

## 優先錄取資格

計畫針對特定身分學生設有優先錄取機制：
- 低收入戶子女
- 中低收入戶子女
- 身心障礙者子女
